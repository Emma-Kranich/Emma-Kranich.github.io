Underwater Adhesive Testing - Chatillon TCD200 Python Script
I was asked to establish serial communication between a computer and the Chatillon machine. The researchers wanted to place their underwater adhesive under the hanging force gauge, use their computer to start the test, have the machine move down at a specific rate while recording the force on the force gauge in an excel file with the time, then stop at a specified distance and return to the initial position. I started by reading the manual to determine the serial communication parameters (baud rate, parity, stop bits, etc.). I then wrote a Python script utilizing Pyserial to establish 2-way communication. I worked with the researchers to set the parameters they wanted and it worked as intended!
